### Группа 1: Виджет "Совещания" на главной странице

#### **TC-BOOKING-001**
**Заголовок:** Базовое отображение виджета "СОВЕЩАНИЯ"  
**Приоритет:** High  
**Предусловие:** Пользователь на главной странице портала

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Найти виджет в левой колонке главной страницы | Отображается блок с заголовком "СОВЕЩАНИЯ" (заглавными буквами) |
| 2 | Проверить наличие календарной сетки | Отображается таблица времени с текущей датой |
| 3 | Проверить отображение времени | Временные слоты от 9:00 до 19:00 |

#### **TC-BOOKING-002**
**Заголовок:** Отображение созданной брони в виджете  
**Приоритет:** High  
**Предусловие:** Пользователь авторизован

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Перейти на страницу бронирования (/135) | Страница загружается |
| 2 | Дважды кликнуть на слот 9:00 в календаре | Открывается форма бронирования |
| 3 | Заполнить поле "Описание": "Тестовая встреча" | Текст отображается в поле |
| 4 | Выбрать "Зал совещаний (к. 218)" из списка комнат | Комната выбирается |
| 5 | Нажать "Сохранить" | Бронь создается, отображается в календаре |
| 6 | Вернуться на главную страницу | Главная страница загружается |
| 7 | Найти виджет "СОВЕЩАНИЯ" | В виджете отображается синий блок с 9:00 до 10:00 |

**Постусловие:** Удалить тестовую бронь

### Группа 3: Работа с бронированиями

#### **TC-BOOKING-013**
**Заголовок:** Открытие формы бронирования  
**Приоритет:** High  
**Предусловие:** Пользователь авторизован, находится на странице бронирования (/135)

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Найти свободный временной слот 10:00 | Слот отображается как доступный |
| 2 | Дважды кликнуть на слот 10:00 | Открывается форма создания бронирования |
| 3 | Проверить поля формы | Отображаются: поле "Описание", выпадающий список "Комната", блок "Период" (время/дата/месяц/год), блок "Повторение" |

#### **TC-BOOKING-014**
**Заголовок:** Просмотр информации о бронировании другого пользователя  
**Приоритет:** Medium  
**Предусловие:** Пользователь авторизован, на странице бронирования (/135) есть брони других пользователей

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Найти занятый слот другого пользователя | Слот отображается синим цветом |
| 2 | Дважды кликнуть на занятый слот | Открывается окно просмотра информации |
| 3 | Проверить отображаемую информацию | Отображаются: автор брони, описание, комната, период времени |

#### **TC-BOOKING-015**
**Заголовок:** Попытка просмотра бронирования неавторизованным пользователем  
**Приоритет:** Medium  
**Предусловие:** Пользователь не авторизован, находится на странице бронирования (/135)

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Найти занятый слот в календаре | Слот отображается синим цветом |
| 2 | Дважды кликнуть на занятый слот | Ничего не происходит, форма не открывается |
| 3 | Проверить доступность создания брони | Двойной клик на свободные слоты также не открывает форму |

#### **TC-BOOKING-016**
**Заголовок:** Отображение формы бронирования  
**Приоритет:** High  
**Предусловие:** Открыта форма бронирования на странице (/135)

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Проверить поле "Описание" | Отображается текстовое поле с плейсхолдером |
| 2 | Проверить список "Комната" | Выпадающий список с вариантами: Зал совещаний (к. 218), Конференц-зал (к. 14), Переговорная (к. 47) |
| 3 | Проверить блок "Период" | Два набора полей: время (9:00-19:00), дата (1-31), месяц (янв-дек), год (2020-2025) |
| 4 | Проверить кнопки управления | Отображаются кнопки: "Сохранить", "Отменить", "Удалить" |

#### **TC-BOOKING-017**
**Заголовок:** Успешное создание бронирования  
**Приоритет:** Critical  
**Предусловие:** Пользователь авторизован, находится на странице бронирования (/135)

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Дважды кликнуть на слот 11:00 | Открывается форма бронирования |

| 3 | В списке "Комната" выбрать "Переговорная (к. 47)" | Значение выбирается |
| 4 | В блоке "Период" установить: 11:00 - 15 - Октябрь - 2025 | Время и дата устанавливаются |
| 5 | Нажать кнопку "Сохранить" | Форма закрывается, в календаре отображается синий блок с 11:00 до 12:00 |
